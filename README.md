# Quiz Application (Java)

## Project Description
A console-based Quiz Application developed using Java. The application presents multiple-choice questions, evaluates answers, calculates scores, displays the final result, and stores quiz results in a text file.

## Features
- User-friendly console interface
- 10 Multiple Choice Questions
- Java Programming and Computer Fundamentals Quiz
- Score Calculation
- Percentage Calculation
- Correct and Incorrect Answer Count
- Input Validation
- Result Storage using File Handling

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- File Handling
- Exception Handling

## How to Run
1. Compile:
   javac src/*.java

2. Run:
   java -cp src Main
